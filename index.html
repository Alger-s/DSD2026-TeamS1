Building a reliable wireless sensing pipeline for motion data.

Team S1 focuses on wireless sensor implementation, communication protocol design, and reliable transmission of motion data in an international collaborative setting.

Team mission

  • Implement and calibrate sensor firmware
  • Design a wireless communication protocol
  • Define packet format, sampling rate, and error handling
  • Complete the IF1 Interface Contract in Sprint 1

Overview

About Team S1

This website summarizes the team background, sprint progress, latest news, and member roles. It is designed as a clear GitHub Pages friendly project site with interactive section switching.

Project scope

Team S1 develops a wireless joint motion capture system. The work combines embedded sensing, protocol definition, firmware integration, and cross-cultural teamwork.

What makes this team special

Hardware-related tasks are mainly handled by China-based members because the physical sensors are currently available there, while planning and collaboration bridge multiple time zones.
