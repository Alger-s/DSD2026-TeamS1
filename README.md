# DSD 2026 Team S1 Website

A static team website for GitHub Pages.

## Sections
- Home
- Progress (Gantt view)
- News
- Members

## Deploy
1. Upload `index.html`, `styles.css`, and `script.js` to the root of your GitHub Pages repository.
2. Commit and push to the default branch.
3. Enable GitHub Pages in repository settings if needed.

## Customization
- Replace role placeholders in the Members section with real names.
- Update task statuses and timeline bars in the Progress section.
- Add latest finished tasks in the News section.
